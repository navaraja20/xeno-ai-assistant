# 🎤 XENO Voice Commands - Complete Guide

XENO now supports **advanced voice commands with spoken feedback**! Simply say "**Hey XENO**" followed by any command below.

## 🌟 Wake Word
- **"Hey XENO"** - Activates voice assistant
- XENO responds: **"Yes Master, how can I help you?"**

## 🚀 Tips for Best Results

1. **Speak clearly** - Normal speaking pace works best
2. **Wait for wake word response** - XENO will say "Yes Master" when ready
3. **Be specific** - Include details in commands (e.g., recipient names, job titles)
4. **Pause briefly** - Short pause after wake word helps accuracy
5. **Check timeline** - Voice commands appear in dashboard activity timeline

## 🛠️ Troubleshooting

**XENO doesn't respond to wake word:**
- Check microphone permissions
- Speak louder or closer to microphone
- Try variations: "Hey Zeno", "XENO"

**Commands not understood:**
- Speak more clearly
- Reduce background noise
- Check internet connection (required for recognition)

**No voice feedback:**
- Check system volume
- Verify TTS is enabled in settings
- Restart XENO
