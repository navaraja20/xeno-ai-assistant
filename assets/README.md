# XENO Assets

This folder contains icon and image assets for XENO.

## Icons Needed

- `XENO.ico` - Windows icon file (for desktop shortcut and taskbar)
- `XENO.png` - PNG icon (for Linux desktop file)
- `XENO.icns` - macOS icon file (for app bundle)

## Creating Icons

You can create custom XENO icons using:
1. Online icon generators (e.g., favicon.io, icon-generator.net)
2. Design tools like GIMP, Photoshop, or Figma
3. AI image generators with prompt: "futuristic AI assistant logo, letter X, cyan blue, minimalist, tech style"

## Recommended Specs

- Size: 256x256px or higher
- Format: PNG with transparency
- Color scheme: Cyan blue (#00d4ff) on dark background
- Style: Modern, tech-focused, gaming aesthetic
