## 📱 Mobile App - Quick Start

### 1. Install Dependencies:
```bash
cd mobile
npm install
```

### 2. Start API Server (in new terminal):
```bash
pip install flask flask-cors pyjwt
python api/mobile_api.py
```

API will run on `http://localhost:5000`

### 3. Configure Mobile App:
Edit `mobile/src/services/XENOAPI.js`:
```javascript
const API_BASE_URL = 'http://YOUR_COMPUTER_IP:5000/api';
// Example: 'http://192.168.1.100:5000/api'
```

Find your IP:
- **Windows**: `ipconfig` (look for IPv4 Address)
- **Mac/Linux**: `ifconfig` (look for inet)

### 4. Run Mobile App:

**Android**:
```bash
npm run android
```

**iOS** (macOS only):
```bash
npm run ios
```

### Mobile Features:
- 📊 Dashboard with quick stats
- 🔔 Push notifications
- 🎤 Voice commands
- ⚡ Quick actions
- ⚙️ Settings

## 📅 Calendar Setup

### First-Time Google Calendar Setup:

1. **Get Credentials**:
   - Go to [Google Cloud Console](https://console.cloud.google.com/)
   - Create a project
   - Enable Google Calendar API
   - Create OAuth 2.0 credentials
   - Download as `calendar_credentials.json`
   - Place in `data/` folder

2. **First Run**:
   - Click "Sync with Google Calendar" in XENO
   - Browser will open for authentication
   - Grant permissions
   - Token saved for future use

3. **Features**:
   - View today's events
   - See upcoming events (7 days)
   - Create new events
   - Conflict detection
   - Voice commands: "What's my schedule?"

## 🛠️ Troubleshooting

### Desktop Issues:

**Calendar not syncing**:
- Check `calendar_credentials.json` in `data/`
- Re-authenticate if needed
- Check internet connection

**Voice commands not working**:
- Ensure microphone permissions
- Check pyttsx3 installation: `pip install pyttsx3`

**Background monitoring not starting**:
- Check console for errors
- Verify module installations

### Mobile Issues:

**Cannot connect to server**:
- Verify API server is running
- Check IP address in XENOAPI.js
- Ensure on same network
- Check firewall settings

**Voice not working**:
- Grant microphone permission
- Check react-native-voice installation

**Notifications not received**:
- Set up Firebase (see mobile/README.md)
- Check notification permissions
- Verify token registration
